How to use FAAST

FAAST offers an intuitive object-oriented interface to handle finite fields and their extensions; see the Reference Manual for a list of available objects and methods. If you want to use FAAST in your programs, simply add to your header files the following lines

#include <faast.hpp>
using namespace FAAST;

The using line is optional, but it will save you a lot of typing.

Assuming you have installed FAAST in $HOME, compile your program with

g++ -I$HOME/include my_program.c -o my_program -L$HOME/lib -lfaast
